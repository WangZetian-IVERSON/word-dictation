# 单词听写系统 — 分享给朋友的方法

---

## 方案 A：给 Windows 朋友（打包成 .exe）

### 1. 安装打包工具
```
venv\Scripts\pip install pyinstaller
```

### 2. 运行打包命令
```
venv\Scripts\pyinstaller --onefile --noconsole --name 单词听写 ^
  --add-data "templates;templates" ^
  --add-data "static;static" ^
  --add-data "data;data" ^
  app.py
```

### 3. 打包结果
生成 `dist\单词听写.exe`，把整个 `dist` 文件夹发给朋友

### 4. 朋友使用方式
双击 `单词听写.exe` → 自动打开浏览器 → 输入密码 `20040311`

> ⚠️ 注意：需要联网（edge-tts 需要访问微软服务器；DeepSeek API 需要联网）

---

## 方案 B：部署到云服务器（手机+电脑都能用，推荐）

### 免费平台：Render.com

1. 注册 https://render.com（免费）
2. 上传代码到 GitHub（先创建仓库）
3. Render 新建 Web Service → 连接 GitHub 仓库
4. 配置：
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn app:app`
5. 部署完成后得到一个网址，如 `https://xxx.onrender.com`
6. 朋友直接浏览器打开这个网址，手机电脑都能用

> ⚠️ 注意：免费版 15 分钟无人访问后会休眠，再次访问需等约 30 秒启动

---

## 方案 C：局域网共享（在家/宿舍内使用）

如果朋友和你在同一个 WiFi 下（比如家庭、宿舍）：

1. 找到你电脑的局域网 IP（在 cmd 输入 `ipconfig`，看 IPv4 地址，如 `192.168.1.5`）
2. 修改 `run.bat`，把服务器改为监听所有网口（`0.0.0.0`）
3. 朋友在浏览器输入 `http://192.168.1.5:5000` 即可访问（包括手机）

---

## 单词本数据位置

所有单词、错题记录在 `data/` 文件夹：
- `data/words.json` — 单词库
- `data/mistakes.json` — 错题本

打包或分享时把 `data/` 文件夹一起带上，朋友就有你的单词库了。
如果不想共享单词，把这两个文件删除或清空即可（第一次使用时会自动创建空文件）。
